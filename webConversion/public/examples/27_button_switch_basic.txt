// SWITCH - Basic Example
// ======================
// A SWITCH button maintains its state when clicked.
//
// How to test:
// 1. I0.0 should be in SWITCH mode (default)
// 2. Click I0.0 - it turns ON (1) and stays ON
// 3. Click again - it turns OFF (0) and stays OFF
// 4. Q0.0 always reflects the state of I0.0

LD I0.0
OUT Q0.0
