// SWITCH - AND Logic (Two Switches in Series)
// =============================================
// Both switches must be ON to activate the output.
//
// How to test:
// 1. I0.0 and I0.1 should be in SWITCH mode
// 2. Turn ON I0.0 - Q0.0 stays OFF
// 3. Turn ON I0.1 - Q0.0 turns ON (both ON)
// 4. Turn OFF either one - Q0.0 turns OFF

LD I0.0
AND I0.1
OUT Q0.0
