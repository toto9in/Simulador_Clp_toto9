// SWITCH - OR Logic (Two Switches in Parallel)
// ==============================================
// Either switch can activate the output.
//
// How to test:
// 1. I0.0 and I0.1 should be in SWITCH mode
// 2. Turn ON I0.0 - Q0.0 turns ON
// 3. Turn OFF I0.0 and turn ON I0.1 - Q0.0 stays ON
// 4. Both OFF - Q0.0 turns OFF

LD I0.0
OR I0.1
OUT Q0.0
